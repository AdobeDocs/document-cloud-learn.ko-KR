# 사용 사례 개요

Acrobat을 사용하여 생산성을 높이고 팀 및 산업을 위해 정보를 실행 가능한 통찰력으로 전환하는 방법에 대해 알아보십시오.

## 업무 라인

다양한 비즈니스 분야의 팀이 Acrobat을 사용하여 일상적인 문서 문제를 해결하고, 워크플로우를 간소화하고, 비즈니스에 중요한 작업을 지원하는 방법을 알아봅니다.

## 새로운 기능

>[!BEGINTABS]

>[!TAB 병목 현상 없이 IT 사용]

[IT 팀](lob/it/it-enablement.md)이 Acrobat Studio를 사용하여 문서 작업 과정을 간소화하고, 보안 및 규정 준수를 강화하고, 조직 전반에 걸쳐 거버넌스 프로그램을 확장하는 방법을 알아봅니다.

>[!TAB Acrobat Studio로 제품 실행 가속화]

마케팅 팀이 Acrobat Studio를 사용하여 단일 PDF 공간에서 [제품 출시](lob/marketing/marketing-product-launch.md) 에셋을 중앙 집중화하고, 관련자 검토를 간소화하고, 조직 준비 상태를 확장하는 방법을 알아봅니다.

>[!ENDTABS]
