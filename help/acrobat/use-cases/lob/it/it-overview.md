# Acrobat IT 사용 사례 개요

IT 팀이 Acrobat Studio를 사용하여 문서 워크플로우를 간소화하고, 보안 및 규정 준수를 강화하고, 조직 전반에 걸쳐 거버넌스 프로그램을 확장하는 방법을 살펴봅니다.

## 새로운 기능

>[!BEGINTABS]

>[!TAB 병목 현상 없이 IT 사용]

[IT 팀이 Acrobat Studio](it-enablement.md)를 사용하여 중요한 문서를 보호하고, 공급업체 계약을 분석하고, 작업 부하를 늘리거나 통제권을 손상시키지 않고 규모에 맞게 거버넌스 정책을 배포하는 방법을 알아봅니다.

>[!ENDTABS]
