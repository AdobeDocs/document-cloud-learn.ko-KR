# 페이지 구성

최적의 고객 경험을 전달하려면 콘텐츠를 빠르게 만들어야 합니다. 원하는 방식으로 페이지를 빠르게 재정렬하는 방법을 살펴볼 수 있습니다. 불연속 페이지를 삽입, 교체, 분할, 재정렬, 회전 및 추출해 전문가다운 PDF을 만들 수 있습니다. 이 비디오 자습서에서는 [새로운 작업 영역 환경](new-workspace.md).
