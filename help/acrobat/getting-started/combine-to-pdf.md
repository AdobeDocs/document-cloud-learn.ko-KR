# 파일을 단일 PDF로 결합

다양한 유형의 파일(PDF, Word, Excel, PowerPoint, JPEG 등)을 결합하는 방법을 알아봅니다. 단일 PDF로 파일을 단일 PDF로 결합하면 다른 사람에게 보내는 컨텐트를 관리하고 제어할 수 있으므로 더 나은 환경을 만들 수 있습니다.
