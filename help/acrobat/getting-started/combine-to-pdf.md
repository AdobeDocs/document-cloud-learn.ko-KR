# 여러 파일을 하나의 PDF으로 결합

다양한 유형의 파일(PDF, Word, Excel, PowerPoint, JPEG 등)을 결합하는 방법을 살펴봅니다. PDF 하나로 통합됩니다. 여러 파일을 하나의 PDF에 결합하면 다른 사용자에게 보내는 내용을 관리하고 제어할 수 있으므로 더 나은 환경을 만들 수 있습니다.
