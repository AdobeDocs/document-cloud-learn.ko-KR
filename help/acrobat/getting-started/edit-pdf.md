# PDF에서 텍스트 편집

PDF에서 단어, 구 또는 전체 페이지를 편집하는 방법을 알아봅니다. Acrobat에서 바로 텍스트를 편집하면 앱들 사이를 오가는 과정 없이 콘텐츠를 정확하고 전문적으로 업데이트할 수 있습니다. 이 비디오 튜토리얼에서는 [새 작업 영역 환경](new-workspace.md).
