# 작업 공간 기본 사항

Acrobat 인터페이스를 통해 일관된 작업 영역 경험을 통해 데스크탑, 웹 및 모바일 디바이스에서 파일과 도구에 손쉽게 액세스하는 방법을 살펴봅니다.
