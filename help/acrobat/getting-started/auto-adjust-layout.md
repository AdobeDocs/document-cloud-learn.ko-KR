# 자동 조정 레이아웃

PDF의 페이지 전체에서 콘텐츠를 리플로우하고 레이아웃을 자동 조정하는 새로운 편집 모드에 대해 알아봅니다. 이 비디오 튜토리얼에서는 [새로운 Acrobat 환경](new-workspace.md)을 사용합니다.

>[!NOTE]
>
>자동 조정 레이아웃이 아직 일부 유형의 PDF 콘텐츠 또는 언어를 지원하지 않습니다.
