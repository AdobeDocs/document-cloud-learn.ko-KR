# 레이아웃 자동 조정

PDF에서 콘텐츠를 리플로우하고 페이지 레이아웃을 자동으로 조정하는 새로운 편집 모드에 대해 알아봅니다. 이 비디오 자습서에서는 [새로운 작업 영역 환경](new-workspace.md).

>[!NOTE]
>
>자동 조정 레이아웃은 아직 모든 유형의 PDF 콘텐츠 또는 언어를 지원하지 않습니다.
