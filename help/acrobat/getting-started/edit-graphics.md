# PDF에서 그래픽 편집

PDF에서 그래픽을 만들고 편집하고 향상시키는 방법을 살펴봅니다. Acrobat에서 바로 그래픽을 편집하면 앱들 사이를 오가는 과정 없이 콘텐츠를 정확하고 전문적으로 업데이트할 수 있습니다. 이 비디오 튜토리얼에서는 [새로운 작업 영역 환경](new-workspace.md)을 사용합니다.
