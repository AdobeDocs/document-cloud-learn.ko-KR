# 실시간으로 공동 작업 수행

어디에서나 원격 작업을 수행할 수 있으므로 신속하고 효율적으로 컨텐츠를 제공하기 위해서는 원활한 공동 작업이 매우 중요합니다. 댓글을 수집하고, 실시간으로 공동 작업하고, 어디서나 진행 상황을 추적하여 프로젝트를 계속 진행하세요. 효율성을 높일 뿐 아니라 콘텐츠 품질도 향상하는 문서 검토 프로세스를 만드는 방법에 대해 알아봅니다. 이 비디오 튜토리얼에서는 [새로운 Acrobat 환경](new-experience.md)을 사용합니다.
