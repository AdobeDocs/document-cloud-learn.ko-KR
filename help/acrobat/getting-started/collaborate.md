# 실시간으로 공동 작업 수행

어디에서나 원격 작업을 수행할 수 있으므로 신속하고 효율적으로 컨텐츠를 제공하기 위해서는 원활한 공동 작업이 매우 중요합니다. 어디서나 실시간으로 의견을 수집하고, 공동 작업에 응답하고, 문서의 진행 상황을 추적하여 프로젝트를 진행하세요. 이 비디오 자습서는 [새로운 작업 영역 환경](new-workspace.md)을 사용합니다.
