# PDF 만들기

[모든 다른 유형의 문서에서 PDF을 만드는 방법](https://www.adobe.com/kr/acrobat/online/convert-pdf.html)을 살펴보세요. Microsoft Office 또는 Creative Cloud 앱에서 만든 파일로 시작하거나 이미지, 스캔 또는 웹 사이트를 이용할 수 있습니다. 콘텐츠를 PDF으로 전환하면 원래 서식을 유지하면서 문서를 편리하고 안정적으로 공유, 보존 및 보호할 수 있습니다. 이 비디오 자습서는 [n개의 환경](new-experience.md)을 사용합니다.

>[!TIP]
>
>Microsoft Office를 항상 사용하십니까? 자주 사용하는 Office 응용 프로그램에서 직접 PDF을 만들 수 있는 [통합](../integrate/integrate-overview.md#microsoft)을 확인하세요.
