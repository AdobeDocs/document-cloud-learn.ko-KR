# 스캔한 문서 최적화

카메라에서 가져온 문서든 스캐너에서 만든 문서든, 더 나은 PDF 보기 및 검색 환경을 위해 Acrobat에서 결과를 향상시키는 방법을 알아보십시오.

>[!TIP]
>
>Acrobat Pro이 있는 경우 Action Wizard 도구에서 [스캔한 문서 최적화]를 실행하여 이 프로세스를 자동화하십시오.
