# 모바일 개요

모든 모바일 장치에서 서명을 위해 문서를 보내고, 전자 서명 진행 상황을 추적하고, 실시간 업데이트를 받을 수 있습니다.

## 새로운 기능

>[!BEGINTABS]

>[!TAB 모바일 친화적 보기]

[모바일 친화적 보기](mobile-friendly.md)를 사용하여 모바일 장치에서 양식을 완성하는 방법을 알아봅니다.

>[!TAB 모바일 친화적 보기 만들기]

개발자 지원 없이 [모바일 친화적](create-mobile-friendly.md) 문서를 원활하게 생성하는 방법을 알아봅니다.
