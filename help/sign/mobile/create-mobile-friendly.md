# 모바일 친화적인 보기 만들기

개발자 지원 없이 모바일 친화적인 문서를 원활하게 생성하는 방법을 알아봅니다.

모바일 보기를 사용하면 발신자는 다음을 활용할 수 있습니다.

* 완료율 증가: 양식 완료율이 높을수록 결과가 향상됨
* 시간 및 비용 절감: 문서를 모바일 친화적인 양식으로 변환하는 데 필요한 작업 방지
* 기술 전문 지식이 필요 없음: 최소한의 노력으로 역동적이고 모바일 친화적인 양식 생성
