# 사본 만들기 사용 시기

이미 계약을 전송했지만 만료되었거나 취소되었거나 다시 전송해야 합니까? 사본 만들기 를 사용하면 기존 계약을 복제하고 변경해야 하는 사항을 조정하고 초 내에 다시 보낼 수 있습니다. 처음부터 다시 빌드할 수 없습니다. 사본 만들기는 다음과 같은 경우에 사용할 수 있습니다.

* 사용자가 계약의 원래 전송자입니다.
* 계약이 최신 서명 요청 환경에서 전송되었습니다.
* 계약에는 감시 수신자 역할이 포함되어 있지 않습니다.
* 문서 보존 또는 GDPR 삭제로 인해 계약이 삭제되지 않았습니다.
* 발신자가 더 이상 액세스할 수 없는 라이브러리 템플릿은 계약에 액세스할 필요가 없습니다.

>[!NOTE]
>
>사본 만들기는 엔터프라이즈 수준 플랜에서 사용할 수 있습니다.
