# 서명자 바꾸기

서명을 위해 문서를 전송할 때 잘못된 전자 메일이 사용된 경우와 같이 서명자를 바꾸는 방법을 알아봅니다.

>[!NOTE]
>
>수신자 바꾸기 의 기본 설정은 현재 수신자를 새 수신자로 완전히 대체합니다. 비즈니스 및 엔터프라이즈 수준 플랜에는 추가 옵션이 있습니다.
