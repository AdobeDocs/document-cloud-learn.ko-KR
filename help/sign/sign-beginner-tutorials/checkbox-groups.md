# 확인란 그룹을 사용하여 필수 선택 사항을 안내합니다.

확인란 그룹을 사용하여 서명자가 양식에서 선택할 수 있는 옵션 수를 제어하는 방법에 대해 알아봅니다. 확인란 그룹은 계약을 제출하기 전에 최소 또는 최대 선택 수 요구와 같은 간단한 규칙을 적용하여 서명자가 문서를 올바르게 완료하도록 하는 데 도움이 됩니다.

확인란을 그룹화하는 방법에 대해 알아보십시오.

* 완료되지 않았거나 잘못된 제출 방지
* 명확한 인라인 피드백으로 서명자 안내
* 양식으로 수집된 정보의 정확성 향상
