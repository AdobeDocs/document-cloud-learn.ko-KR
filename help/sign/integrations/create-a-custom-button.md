# Salesforce용 Adobe Sign에서 사용자 정의 단추 만들기

Salesforce.com에서 표준 개체에 사용자 정의 단추를 만드는 방법을 알아봅니다. 이 예제에서는 사용자 정의 버튼이 **연락처** Salesforce에 로그인합니다. 단추를 클릭하면 전송 프로세스가 시작되고 **연락처** 계약서에 직접 서명
