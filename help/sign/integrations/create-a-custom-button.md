# Acrobat Sign에서 나만의 버튼 만들기 [!DNL Salesforce]

Salesforce.com에서 표준 개체에 사용자 정의 단추를 만드는 방법을 알아봅니다. 이 예제에서는 사용자 지정 단추가 **연락처** 아이콘을 [!DNL Salesforce]. 단추를 클릭하면 전송 프로세스가 시작되고 **연락처** 계약서에 직접 입력.
