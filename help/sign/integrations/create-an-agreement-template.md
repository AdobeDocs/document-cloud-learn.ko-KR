# 문서 작성기 [!DNL Salesforce]

Salesforce용 문서 빌더를 사용하여 재사용 가능한 문서 템플릿을 만드는 방법을 알아봅니다. 재사용 가능한 문서 템플릿은 서명 트랜잭션의 속도와 일관성을 제공하고 문서 수정 및 변경 사항을 관리할 수 있는 기능을 제공합니다.
